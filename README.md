# CS202_22APCS1_Group9

## How to run

### Pre-requisites

 - C++
 - CMake (version 3.16)

### Build

- Clone this repository.
- Run the following command:
  - `mkdir build`
  - `cmake -S . -B build`
  - `cmake -C build`
- The executable file will be in the `build` folder.

### Run
- `./build/bin/CS202_Group9`
